# Additive Monitoring System Prototype
##Description
A prototype for an additive consumption monitoring system designed to measure and visualize real-time material usage. The system uses load cells to track weight changes and quantify material consumption over time. This data is processed and displayed on a live graph for monitoring and detection of dosing inconsistencies, where deviations from target percentages can result in significant material overuse. Real-time monitoring enables timely intervention and correction of such issues.

##Hardware
- Load Cells
- HX711 Amplifiers
- ESP32 Microcontroller
- Electrical connections (wiring and soldered connections)
- frame
